
## Laravel Vue Structure

### Build Setup

``` bash
# Requirement:
    1. composer
    2. node && npm
    
# Pre develop
    1. composer install
    2. npm install
    3. Create .env file and run php artisan key:generate
    
# Run server Laravel
    1. You can use laragon, xamp, etc.... for your local development.
    2. You can use php artisan serve

# Run single page app for develop:
    $ npm run watch

# build for production and launch server
    $ npm run build
```
